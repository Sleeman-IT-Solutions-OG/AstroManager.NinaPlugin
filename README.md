# AstroManager.NinaPlugin

Official public repository for the AstroManager plugin for N.I.N.A.

## Purpose

AstroManager.NinaPlugin connects N.I.N.A. with AstroManager and extends the imaging workflow with AstroManager features and services.

Learn more about AstroManager here:

https://astro.sleeman.at/astro-manager

## Installation

The plugin can be installed:

- through the official N.I.N.A. plugin repository
- through the releases published in this GitHub repository

## Repository Contents

This repository contains the source code for the public AstroManager N.I.N.A. plugin.
